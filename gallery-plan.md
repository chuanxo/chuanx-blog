# 照片图册功能规划

## 目标
在现有 Hugo 博客中添加一个照片图册功能，用于展示照片。

## 需求确认 ✅

| 需求项 | 确认结果 |
|--------|---------|
| 存储方式 | 外部图床 |
| 照片数量 | 1000张以内 |
| 展示布局 | 网格布局（整齐美观） |
| 分类方式 | 按更新时间、按图册名字 |
| 交互功能 | 点击放大 |

---

## 实现方案

### 技术选型

| 组件 | 选择 | 说明 |
|------|------|------|
| 图片存储 | 外部图床 | 待确认具体服务 |
| 展示布局 | CSS Grid | 响应式网格 |
| 灯箱效果 | PhotoSwipe / lightGallery | 点击放大浏览 |
| 相册管理 | Hugo content + YAML data | 灵活管理相册和照片信息 |

### 目录结构设计

```
content/
├── gallery/
│   ├── _index.md          # 图册首页（显示所有相册）
│   ├── travel/            # 旅行相册
│   │   ├── index.md
│   │   └── photos.yaml    # 照片列表
│   ├── life/              # 生活相册
│   │   ├── index.md
│   │   └── photos.yaml
│   └── ...
```

### 相册数据格式 (photos.yaml)

```yaml
photos:
  - url: "https://your-cdn.com/photos/photo1.jpg"
    title: "照片标题"
    date: 2026-02-12
  - url: "https://your-cdn.com/photos/photo2.jpg"
    title: "另一张照片"
    date: 2026-02-10
```

## 外部图床选择

请选择你想要使用的图床服务：

| 图床服务 | 优点 | 缺点 | 费用 |
|---------|------|------|------|
| **阿里云 OSS** | 国内速度快，稳定 | 配置稍复杂 | 按量付费，很便宜 |
| **七牛云** | 有免费额度，CDN加速 | 免费额度有限 | 10G免费，超出付费 |
| **又拍云** | 有免费联盟计划 | 需要备案 | 可免费 |
| **腾讯云 COS** | 国内速度快 | 配置复杂 | 按量付费 |
| **Cloudflare R2** | 无出站流量费 | 国内速度可能慢 | 10G免费 |
| **GitHub + jsDelivr** | 免费，简单 | 国内访问不稳定 | 免费 |

**你已经有图床服务了吗？还是需要我推荐一个？**
