# 关于本站

欢迎来到我的个人博客！本站使用以下技术构建：

- **[Hugo](https://gohugo.io/)** - 世界上最快的静态网站生成器
- **[PaperMod](https://github.com/adityatelange/hugo-PaperMod)** - 快速、简洁、响应式的 Hugo 主题
- **[GitHub Pages](https://pages.github.com/)** - 免费托管与自动部署
- **[Giscus](https://giscus.app/)** - 基于 GitHub Discussions 的评论系统

## 功能特性

- 全文搜索功能
- GitHub Discussions 评论系统
- Google Analytics 4 网站分析
- 深色/浅色模式切换
- 完全响应式设计
- 极快的页面加载速度
